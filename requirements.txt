🛠️ 快速开始 (Quick Start)

1. 环境准备 (Installation)
请确保本地已安装 Python 3.8+。建议使用虚拟环境（如 .venv）来隔离项目依赖。
在项目根目录下，打开终端并运行以下命令，一键安装所有核心依赖：

Bash
pip install -r requirements.txt
2. 启动系统 (Run)
环境安装完毕后，在终端中运行主程序，即可唤醒可视化控制面板：

Bash
python main.py
3. 标准操作流 (Workflow)
基础设置：选择包含经纬度数据的业务 Excel 表格，设置目标保存目录与渲染质量。
底图与网络：选择心仪的底图风格与缓存精度，点击雷达面板的**“获取/补全图源”**将底图高速缓存至本地硬盘（可随时安全急停）。
视觉与算法：微调红黄灵敏度，开启/关闭需要的视觉图层（如行政区划边界、地铁等），并按需调整“高定边界”参数。
一键执行：点击左下角常驻的 🚀 开始执行空间渲染，等待进度条完成，即可在目标文件夹获取高清成图。
🔒 隐私与安全 (Privacy & Security)
本系统承诺：完全本地化运行。用户的业务坐标数据（Excel）、分析过程、生图结果均在本地内存和硬盘中处理与销毁，绝不触网上传。离线地图底图拉取完毕后支持完全断网调用，最高级别保障商业数据的安全与隐私。
